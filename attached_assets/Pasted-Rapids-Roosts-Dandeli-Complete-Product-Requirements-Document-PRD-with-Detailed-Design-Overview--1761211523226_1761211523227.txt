Rapids Roosts Dandeli - Complete Product Requirements Document (PRD) with Detailed Design
Overview
A modern, engaging, partially automated tour booking web app featuring an immersive nature-inspired UI enhanced with parallax animations. It delivers seamless booking, customer status tracking, and admin manual confirmation with a focus on cost-effective communication using email for customers and WhatsApp notifications for admins.

Page-wise Design Details with Parallax Animation
1. Home Page
Hero Section: Fullscreen looping drone video banner with parallax layers (forest canopy, river, mist). Headline overlay with typed-effect tagline: "Explore the Wild Heart of Karnataka."

Search & Booking CTA: Prominent booking quick search (date, activity, guests) with gentle floating animation.

Adventure Highlights: Horizontally scrolling parallax cards detailing rafting, safari, etc., with icon hover zoom+shadow.

Footer: Social media, contact, quick links with fade-in on scroll.

Parallax Effects: Multi-depth scene scroll for immersive UX, smooth horizontal scroll on adventure cards.

2. Booking Page
Multi-Step Responsive Form:
Step indicators with progress bar and glowing forest-green accents.
Input fields with river-blue focus rings and validation tooltips.

Sidebar Booking Summary: Parallax offset as the user scrolls form.

Sticky Dynamic "Book Now" Button: Pulsates when form valid.

Animations:
Slide transitions between steps, input focus glows, content fades.

3. Booking Acknowledgement Page
Centered Confirmation Card: Booking ID displayed with "Pending Confirmation" badge (earthy brown).

Background: Layered subtle ripple effect simulating water, parallax scroll effect with slow-moving wave layers.

Prompt to Check Status: Button with gradient teal-blue hover.

Animations:
Badge pulse every 3 seconds, water ripple continuous animation.

User Journey & Functional Flow
Step	Description
Booking Initiation	User inputs booking details via multi-step form on booking page.
Data Submission	Data sent to backend and workflow automation (n8n webhook).
Customer Acknowledgment	Immediate email sent to user with Booking ID and pending status.
Admin Notification	Booking details sent via WhatsApp message to admin with action options.
Manual Booking Confirmation	Admin responds (manually via WhatsApp or admin panel). Booking updated.
Status Tracking	User checks booking status anytime on web using Booking ID.
Booking Confirmation	On confirmation, system sends final email, optional WhatsApp message.

Summary
The Rapids Roosts Dandeli Web App is a modern, visually appealing platform designed to create an immersive nature adventure booking experience enriched with elegant parallax animation effects. The booking process is partially automated for a smooth operational workflow that combines cost-efficiency with usability. Users enjoy immediate acknowledgments, seamless status tracking, and an engaging UI while admins get efficient notification and action flows.
